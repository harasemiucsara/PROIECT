# proiect - PIU

**GESTIONARE STOC PRODUSE**

Aplicația este un sistem de gestiune a stocurilor destinat administrării produselor din 
categoria suplimentelor alimentare și a produselor de îngrijire personală. Scopul aplicației 
este de a facilita organizarea, monitorizarea și actualizarea stocului de produse într-un mod 
eficient și ușor de utilizat.

Prin intermediul aplicației, utilizatorul poate adăuga produse noi în baza de date, poate 
modifica informațiile existente și poate elimina produsele care nu mai sunt disponibile. 
Pentru fiecare produs sunt stocate detalii relevante, precum denumirea, categoria, cantitatea 
disponibilă în stoc, prețul și alte informații utile pentru gestionarea corectă a inventarului.
